Avro Python API Documentation

Overview​

This documentation provides details for classes, functions, and methods found in the avro.io, avro.datafile, and avro.schema modules. It is intended for Python developers who want to use Apache Avro for serialization, deserialization, and schema management.

Module: avro.io​

Class: BinaryEncoder​

Encodes Avro objects into binary format and writes them to an output stream.

Example​

from avro.io import BinaryEncoder
import io

buffer = io.BytesIO()
encoder = BinaryEncoder(buffer)
encoder.write_int(42)
encoder.write_utf8("hello")

Methods​

__init__(writer)​

Parameters

  • writer (io.BufferedWriter or similar): The writable output stream.

Writes a null value.
Returns: None
Raises: None

Writes a boolean value.
Parameters:

  • value (bool): Boolean value to encode.

write_int(value)​

Writes a 32-bit integer using variable-length zig-zag encoding.
Parameters:

  • value (int): Integer to encode.

write_long(value)​

Writes a 64-bit long using variable-length zig-zag encoding.
Parameters:

    Raises: AvroTypeException.

Writes a 32-bit floating-point value.
Parameters:

  • value (float): Float value to encode.
    Raises: AvroTypeException.

Writes a 64-bit floating-point value.
Parameters:

  • value (float): Double value to encode.

Writes byte data (length-prefixed).

write_utf8(value)​

Class: BinaryDecoder​

Decodes Avro objects from a binary stream.

Example​

from avro.io import BinaryDecoder
import io

buffer = io.BytesIO(b'\x84\x02hello')
decoder = BinaryDecoder(buffer)
number = decoder.read_int()
text = decoder.read_utf8()

Module: avro.datafile​

Class: DataFileWriter​

Writes Avro objects to a file using a specified schema and codec.

Example​

import io, avro.schema
from avro.datafile import DataFileWriter
from avro.io import DatumWriter

schema = avro.schema.parse(open("user.avsc").read())
buffer = io.BytesIO()

writer = DataFileWriter(buffer, DatumWriter(), schema)
writer.append({"name": "Alyssa", "favorite_number": 256})
writer.flush()
writer.close()

Methods​

__init__(writer, datum_writer, writers_schema=None, codec='null')​

Parameters:

  • writer (file or io.BufferedWriter): Output file object.
  • datum_writer (avro.io.DatumWriter): Used to serialize data.
  • writers_schema (avro.schema.Schema, optional): Schema for writing data.
  • codec (str): Compression codec ('null', 'deflate', 'snappy').
    Raises: AvroTypeException for invalid schema.
append(datum)​

Writes a datum (Avro record) to the file.
Parameters:

  • datum (dict): Data matching the writer’s schema.
    Raises: AvroTypeException if datum does not conform to schema.
flush()​

Flushes buffered data to disk.
Returns: None

close()​

Closes the writer and file stream.
Returns: None

Class: DataFileReader​

Reads Avro objects from a file.

Example​

from avro.datafile import DataFileReader
from avro.io import DatumReader

reader = DataFileReader(open("users.avro", "rb"), DatumReader())
for user in reader:
    print(user)
reader.close()

Methods​

__init__(reader, datum_reader)​

Parameters:

  • reader (file or io.BufferedReader): Input file object.
  • datum_reader (avro.io.DatumReader): Used to deserialize data.

Returns an iterator over data records.
Returns: Iterator of deserialized records.

close()​

Closes the reader and file stream.
Returns: None

Module: avro.schema​

Class: Schema​

Base class for Avro schemas.

Attributes​

  • type (str): The schema type (record, enum, fixed, etc.).
  • props (dict): Additional schema properties.

Example​

import avro.schema
schema = avro.schema.parse(open("user.avsc").read())
print(schema.type)

Function: parse(json_string)​

Parses a JSON string into a Schema object.

Parameters:

  • json_string (str): JSON representation of a schema.

Returns:

  • Schema object

Raises:

  • SchemaParseException: Raised when schema parsing fails.

Example​

import avro.schema
schema = avro.schema.parse('{"type": "string"}')

Data Encoding & Decoding Flow​

flowchart LR
    A[User Data] --> B[DatumWriter]
    B --> C[BinaryEncoder]
    C --> D[DataFileWriter]
    D --> E[Avro File]
    E --> F[DataFileReader]
    F --> G[BinaryDecoder]
    G --> H[DatumReader]
    H --> I[Python Object]

Exceptions Reference​

  • AvroTypeException: Raised when data does not match schema.
  • SchemaParseException: Raised when schema parsing fails.
  • EOFError: Raised when reading beyond end of stream.
