OpenWeatherMap API — Current Weather Data

Get real-time weather data for any city worldwide using the OpenWeatherMap API.

Authentication​

To use this API, you need an API key.

  • Base URL: https://api.openweathermap.org/data/2.5/
  • API Key Parameter: appid=YOUR_API_KEY

Endpoint: Get Current Weather​

GET /weather

Returns current weather data for a specific city or coordinates.

Query Parameters​

NameTypeRequiredDescription
qstringNoCity name (e.g. London)
latfloatNoLatitude (use with lon)
lonfloatNoLongitude (use with lat)
appidstringYesYour API key
unitsstringNoUnits of measurement (metric, imperial)
langstringNoLanguage code (e.g. en, es, fr)

Example Request​

GET /weather?q=London&units=metric&appid=YOUR_API_KEY
Host: api.openweathermap.org

Example Response (200 OK)​

{
  "coord": { "lon": -0.13, "lat": 51.51 },
  "weather": [{ "main": "Clouds", "description": "overcast clouds" }],
  "main": {
    "temp": 18.75,
    "feels_like": 17.3,
    "humidity": 77
  },
  "name": "London"
}

Response Schema​

FieldTypeDescription
coord.lonfloatLongitude of the location
coord.latfloatLatitude of the location
weather[0].mainstringGroup of weather parameters (Rain, etc.)
weather[0].descriptionstringWeather condition description
main.tempfloatCurrent temperature
main.feels_likefloatPerceived temperature
main.humidityintegerHumidity percentage
namestringName of the city

Endpoint: 5-Day Forecast​

GET /forecast

Get 5-day / 3-hour forecast weather data.

Example:​

GET /forecast?q=London&units=metric&appid=YOUR_API_KEY

See full docs

Endpoint: Air Pollution Data​

GET /air_pollution

Provides current, forecast, and historical air pollution data.

Example:​

GET /air_pollution?lat=51.51&lon=-0.13&appid=YOUR_API_KEY

See full docs

Code Examples​

Python (using requests)​

import requests

url = "https://api.openweathermap.org/data/2.5/weather"
params = {
    "q": "London",
    "units": "metric",
    "appid": "YOUR_API_KEY"
}

response = requests.get(url, params=params)
print(response.json())

JavaScript (using fetch)​

const apiKey = "YOUR_API_KEY";
const city = "London";
fetch(`https://api.openweathermap.org/data/2.5/weather?q=${city}&units=metric&appid=${apiKey}`)
  .then(response => response.json())
  .then(data => console.log(data));

curl​

curl "https://api.openweathermap.org/data/2.5/weather?q=London&units=metric&appid=YOUR_API_KEY"

Error Responses​

CodeMessageDescription
401Invalid API keyCheck your API key
404City not foundInvalid city or coordinates
