Login API

Authenticate a user and receive a JWT token for future API access.

POST /api/login​

Authenticates a user with their credentials.

🧾 Headers​

NameTypeRequiredDescription
Content-TypestringYesMust be application/json

📤 Request Body​

{
  "username": "johndoe",
  "password": "yourpassword"
}
FieldTypeRequiredDescription
usernamestringYesYour account username/email
passwordstringYesYour account password

📥 Response​

{
  "token": "eyJhbGciOiJIUzI1NiIsInR...",
  "expiresIn": 3600
}
FieldTypeDescription
tokenstringJWT token for future API requests
expiresInnumberToken validity in seconds

📊 Response Codes​

CodeMeaning
200Successful authentication
401Invalid credentials
500Server error

🧪 Example Request with curl​

curl -X POST https://api.example.com/login \
  -H "Content-Type: application/json" \
  -d '{"username": "johndoe", "password": "yourpassword"}'

💻 Example in JavaScript​

fetch('https://api.example.com/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ username: 'johndoe', password: 'yourpassword' })
})
  .then(res => res.json())
  .then(data => console.log('Token:', data.token));
